## What this repo is for

This repo is for public-facing Butler Systems research notes.

It is meant to hold:

- exact theorem notes
- bounded release indexes
- controlled public research drops

## What this repo is not for

This repo is not the full private nest.

It does not attempt to publish, all at once:

- broader internal chronology
- unfinished event dynamics
- large witness swarms as substitutes for proof
- full canopy ontology as a novelty claim

## Release discipline

Each note is released with an explicit boundary between:

- what is claimed
- what is withheld
- what remains open

That is the working rule of the Butler Systems ladder.
